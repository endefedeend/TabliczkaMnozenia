# TabliczkaMnozenia
edukacyjna tabliczka mnożenia  w Cpp i Raylib

Autor zezwala na bezpłatne używanie, kopiowanie,
modyfikowanie i rozpowszechnianie programu
w celach edukacyjnych bez konieczności uzyskiwania
dodatkowej zgody.

Program został napisany w języku C++ przy użyciu biblioteki raylib 6.0.
